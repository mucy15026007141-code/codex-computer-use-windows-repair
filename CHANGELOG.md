## [0.1.0] - 2026-07-14

### Added

- Dynamic runtime content-hash calculation.
- Read-only runtime diagnostics.
- Explicit `-Apply` repair mode.
- Windows PowerShell 5.1 and long-path support.
- Same-volume staging, checksum validation, and atomic directory promotion.

### Verified
